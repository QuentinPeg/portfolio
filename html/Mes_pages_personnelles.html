Mes Projets Personnelles

Personal Banking

Application de gestion de compte. Cette application permet de faire vos comptes et de Visualiser vos dépenses de manière simple. Vous pouvez aussi vous voir vos envellopes de budget.

Langages utilisées : React + Vite + Tailwind + Supabase

Temps de codes : ~ 300 heures

Jarvis

Projet de création d'un assistant personnel basé sur l'Intelligence Artificielle. Cette application permet de gérer les tâches quotidiennes et de répondre à vos questions. Elle tourne sur un raspberry pi 5, me permet de controller mon ordinateur par la voix.

Notions abordées : Python, Intelligence Artificielle

Temps consacrés : ~ 100 heures

Homelab Personelle

Projet de mise en place d'un lab domestique pour gérer mon réseau personnelle. Présence de d'outils de streaming, mais aussi d'IA personnelle.

Notions utilisées : Conteneurisation, gestion réseaux, gestion de flux vidéos

Temps de consacrés : ~ 500 heures
